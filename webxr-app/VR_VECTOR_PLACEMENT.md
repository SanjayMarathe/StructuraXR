# VR Vector Placement - Precise Two-Point System

## ðŸŽ¯ How It Works

The VR vector placement system allows you to precisely create force vectors by specifying:
1. **Start Point** - Where the vector begins
2. **End Point** - Where the vector ends
3. **Depth Control** - Adjust how far the point is from you

## ðŸ“‹ Step-by-Step Guide

### Step 1: Set Start Point
1. **Point your controller** at where you want the vector to start
2. **Adjust depth** using thumbstick (forward/back) or A/B buttons
3. **Press Trigger** - Green sphere appears at start point

### Step 2: Set End Point
1. **Point your controller** at where you want the vector to end
2. **Adjust depth** using thumbstick (forward/back) or A/B buttons
3. **Press Trigger** - Red sphere appears at end point, vector arrow is created

### Step 3: Vector Created
- Vector arrow appears from start (green) to end (red)
- Vector magnitude is automatically calculated from distance
- Vector direction points from start to end

- **Final Arrow** - Confirmed vector with ID label

## ðŸ’¡ Tips

- **Depth Range**: 0.5m to 10m from controller
- **Precise Placement**: Use thumbstick for fine depth adjustments

- **Cancel Anytime**: Press Grip to cancel and start over

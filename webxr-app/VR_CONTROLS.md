### Force Vector Placement (Precise Two-Point System)

**To Create a Force Vector:**
1. **Point controller** at where you want the vector to START
2. **Adjust depth** using thumbstick (forward/back) or A/B buttons
3. **Press Trigger** - Green sphere appears at start point
4. **Point controller** at where you want the vector to END
5. **Adjust depth** using thumbstick (forward/back) or A/B buttons
6. **Press Trigger again** - Red sphere appears, vector is created
7. **Press Grip** - Cancels vector placement at any time

**Visual Indicators:**
- **Green Sphere** - Start point marker
- **Red Sphere** - End point marker
- **Cyan Ring** - Depth indicator (shows current depth position)
- **Red Arrow** - Temporary vector preview

**Depth Control:**
- **Thumbstick Forward** (push away) - Increase depth (move point further)
- **Thumbstick Backward** (pull toward) - Decrease depth (move point closer)
- **A/X Button** - Increase depth
- **B/Y Button** - Decrease depth
- **Depth Range**: 0.5m to 10m from controller

### Object Interaction

**Grab and Move:**
- **Point controller** at object
- **Press Trigger** - Grabs object
- **Move controller** - Object follows
- **Release Trigger** - Drops object

### UI Panel Interaction

**Floating Panels:**
- **Point controller** at floating UI panel
- **Press Trigger** - Clicks button/input (same as mouse click)
- Panels are positioned at comfortable viewing angles
- Left panel: Main controls (always visible)
- Right panel: Analysis panel (opens when needed)
